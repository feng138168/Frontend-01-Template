## 简单语句
- ExpressionStatement   a = 1 + 2;
- EmptyStatement    ;
- DebuggerStatement     debugger;
- ThrowStatement    throw a;
- ContinueStatement     continue label1;
- BreakStatement    break label2;
- ReturnStatement   return 1 + 2;
